# Story 4.3: Monitor de GPU NVIDIA em Tempo Real

## Metadata
- **Epic:** 4 - System Monitoring Dashboard
- **Status:** Draft
- **Priority:** P0
- **Story Points:** 5
- **Depends On:** 4.1, 3.1
- **FR References:** FR-042, FR-045, NFR-009

## User Story
Como usuario com GPU NVIDIA, eu quero ver uso de GPU, VRAM, clock speed e temperatura em tempo real, para que eu monitore a performance da placa durante gaming ou workloads pesados.

## Description
Expandir o monitor loop da Story 4.1 com metricas de GPU NVIDIA via NVML (reutilizando `NvmlHandle` da Story 3.1). Adicionar `GpuMetrics` ao `SystemMetrics`. Se NVML nao disponivel, GPU metrics sao omitidas graciosamente.

Tambem adicionar temperatura da GPU ao monitor — NVML tem API direta para temperatura (`nvmlDeviceGetTemperature`).

## Acceptance Criteria
- [ ] AC-1: Evento `"system-metrics"` inclui campo `gpu: Option<GpuMetrics>` (`None` se NVML indisponivel)
- [ ] AC-2: `GpuMetrics` contem: `usage_percent: f32`, `memory_used_bytes: u64`, `memory_total_bytes: u64`, `core_clock_mhz: u32`, `memory_clock_mhz: u32`, `temperature_celsius: u32`
- [ ] AC-3: Metricas via `nvml-wrapper`: `device.utilization_rates()`, `device.memory_info()`, `device.clock(ClockType::Graphics)`, `device.temperature(TemperatureSensor::Gpu)`
- [ ] AC-4: Se NVML nao disponivel, `gpu: None` — UI exibe "GPU NVIDIA nao disponivel" sem erro
- [ ] AC-5: Dashboard exibe: uso de GPU%, VRAM usada/total, temperatura com indicador de cor
- [ ] AC-6: Temperatura GPU: verde < 70C, amarelo 70-85C, vermelho > 85C (threshold configuravel via AppConfig)
- [ ] AC-7: Historico de GPU metrics armazenado no `metricsStore` (30 amostras)

### nvml-wrapper API
```rust
use nvml_wrapper::{Nvml, enum_wrappers::device::{Clock, TemperatureSensor}};

fn build_gpu_metrics(nvml: &Nvml) -> Option<GpuMetrics> {
    let device = nvml.device_by_index(0).ok()?;

    let util = device.utilization_rates().ok()?;
    let mem = device.memory_info().ok()?;
    let core_clock = device.clock(Clock::Graphics, nvml_wrapper::enum_wrappers::device::ClockId::Current).ok()?;
    let mem_clock = device.clock(Clock::Memory, nvml_wrapper::enum_wrappers::device::ClockId::Current).ok()?;
    let temp = device.temperature(TemperatureSensor::Gpu).ok()?;
    let name = device.name().ok().unwrap_or_default();

    Some(GpuMetrics {
        device_name: name,
        usage_percent: util.gpu as f32,
        memory_used_bytes: mem.used,
        memory_total_bytes: mem.total,
        core_clock_mhz: core_clock,
        memory_clock_mhz: mem_clock,
        temperature_celsius: temp,
        memory_usage_percent: (mem.used as f32 / mem.total as f32) * 100.0,
    })
}
```

### Acesso ao NvmlHandle no monitor loop
O `NvmlHandle` esta em `AppState` (Story 3.1). O monitor loop recebe `AppHandle` — acessar o state via `app.state::<AppState>()`.

### Thresholds de temperatura
Usar `AppConfig.thresholds.gpu_celsius` (default 90C — Story 1.3) para colorir o indicador de temperatura.
