# Story 4.4: Monitoramento de Temperatura CPU com Alertas

## Metadata
- **Epic:** 4 - System Monitoring Dashboard

## User Story
Como usuario do Windows Optimizer, eu quero ver a temperatura da CPU em tempo real com alertas visuais quando ultrapassar thresholds, para que eu identifique rapidamente problemas de superaquecimento.

## Description
Implementar monitoramento de temperatura da CPU em `platform/thermal.rs`. Estrategia com fallback (CON-007): tentativa via WMI `MSAcpi_ThermalZoneTemperature` (requer admin) com fallback para `sysinfo` component temperatures. Adicionar `ThermalMetrics` ao `SystemMetrics`. Implementar sistema de alertas visuais no frontend quando temperatura ultrapassa threshold configurado.

## Technical Notes

### platform/thermal.rs
```rust
#[derive(Debug, Clone, serde::Serialize)]
pub enum ThermalSource {
    WmiAcpi,
    Sysinfo,
    Unavailable,
}

#[derive(Debug, Clone, serde::Serialize)]
pub struct ThermalMetrics {
    pub cpu_celsius: Option<f32>,
    pub source: ThermalSource,
}

pub fn read_cpu_temperature(wmi: &WmiClient, sys: &System) -> ThermalMetrics {
    // 1. Tentar WMI MSAcpi_ThermalZoneTemperature
    if let Ok(temps) = wmi.query_thermal_zones() {
        if let Some(temp) = temps.first() {
            // WMI retorna temperatura em decikelvin: (temp - 2732) / 10.0
            let celsius = (*temp as f32 - 2732.0) / 10.0;
            if celsius > 0.0 && celsius < 150.0 { // sanity check
                return ThermalMetrics {
                    cpu_celsius: Some(celsius),
                    source: ThermalSource::WmiAcpi,
                };
            }
        }
    }

    // 2. Fallback: sysinfo components
    for component in sys.components() {
        if component.label().to_lowercase().contains("cpu") {
            return ThermalMetrics {
                cpu_celsius: Some(component.temperature()),
                source: ThermalSource::Sysinfo,
            };
        }
    }

    // 3. Indisponivel
    ThermalMetrics {
        cpu_celsius: None,
        source: ThermalSource::Unavailable,
    }
}
```

### WMI ThermalZone query
```rust
// Em platform/wmi_client.rs
#[derive(serde::Deserialize)]
struct ThermalZone {
    #[serde(rename = "CurrentTemperature")]
    current_temperature: u32, // em decikelvin
}

pub fn query_thermal_zones(&self) -> Result<Vec<u32>, AppError> {
    let results: Vec<ThermalZone> = self.connection.raw_query(
        "SELECT CurrentTemperature FROM MSAcpi_ThermalZoneTemperature"
    )?;
    Ok(results.iter().map(|t| t.current_temperature).collect())
}
```

### Nota sobre WMI e admin (CON-007)
`MSAcpi_ThermalZoneTemperature` frequentemente requer admin e pode nao funcionar em todo hardware. O fallback via sysinfo e importante para a maioria dos usuarios. Logar a fonte usada: `tracing::info!("CPU temperature source: {:?}", metrics.source)`.
