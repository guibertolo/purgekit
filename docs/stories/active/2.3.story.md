# Story 2.3: Limpeza de Windows Update Cache, Prefetch e Recycle Bin

## Metadata

## User Story
Como usuario do Windows Optimizer, eu quero limpar Windows Update cache, Prefetch e esvaziar a Lixeira, para que eu recupere espaco em disco de arquivos desnecessarios do sistema.

## Description
Implementar scan e limpeza de tres categorias adicionais que requerem atencao especial:

1. **Windows Update Cache** (`%WINDIR%\SoftwareDistribution\Download\`) â€” requer admin. Se o servico Windows Update estiver rodando, avisar e oferecer parar temporariamente.
2. **Prefetch** (`%WINDIR%\Prefetch\`) â€” requer admin. Avisar que primeiro boot apos limpeza pode ser mais lento.
3. **Recycle Bin** â€” sem elevacao. Usa Shell API `SHEmptyRecycleBin`. Exibe tamanho total da lixeira.

## Acceptance Criteria
- [ ] AC-1: Scan de "Windows Update Cache" retorna tamanho de `%WINDIR%\SoftwareDistribution\Download\` com `requires_elevation: true`
- [ ] AC-2: Se servico "wuauserv" (Windows Update) estiver rodando ao tentar limpar, retorna aviso `update_service_running: true`
- [ ] AC-3: Tauri command `stop_windows_update_service` para e reinicia o servico wuauserv (requer elevacao)
- [ ] AC-4: Scan de "Prefetch" retorna tamanho de `%WINDIR%\Prefetch\` com `requires_elevation: true`
- [ ] AC-5: Limpeza de Prefetch inclui warning `first_boot_slower: true` no resultado
- [ ] AC-6: Scan de "Recycle Bin" retorna tamanho total da lixeira via `SHQueryRecycleBin` API
- [ ] AC-7: Limpeza de Recycle Bin usa `SHEmptyRecycleBin` API (nao delecao manual de arquivos)
- [ ] AC-8: Recycle Bin scan e limpeza nao requerem elevacao admin
- [ ] AC-9: Todas as tres categorias aparecem no `ScanResult` do `scan_system_cache` command

### Recycle Bin via Shell API
```rust
use windows::Win32::UI::Shell::{SHEmptyRecycleBinW, SHQueryRecycleBinW, SHQUERYRBINFO};
use windows::Win32::UI::Shell::SHERB_NOCONFIRMATION;

pub fn get_recycle_bin_size() -> Result<u64, AppError> {
    unsafe {
        let mut info = SHQUERYRBINFO {
            cbSize: std::mem::size_of::<SHQUERYRBINFO>() as u32,
            ..Default::default()
        };
        SHQueryRecycleBinW(None, &mut info)?;
        Ok(info.i64Size as u64)
    }
}

pub fn empty_recycle_bin() -> Result<(), AppError> {
    unsafe {
        SHEmptyRecycleBinW(None, None, SHERB_NOCONFIRMATION)?;
        Ok(())
    }
}
```
