# Story 1.2: Dark Theme e UI Shell

## Metadata
- **Epic:** 1 - Foundation & Tauri Project Setup
- **Status:** Ready for Review
- **Priority:** P0
- **Story Points:** 5
- **Depends On:** 1.1
- **FR References:** NFR-003, NFR-004, NFR-012

## User Story
Como usuario do Windows Optimizer, eu quero uma interface dark theme moderna com sidebar de navegacao e area de conteudo, para que eu possa navegar entre os modulos do app de forma intuitiva.

## Description
Implementar o layout shell completo do app: dark theme Tailwind com cores definidas, sidebar fixa de navegacao com links para todos os modulos do MVP, area de conteudo dinamica com TanStack Router, e componentes base de feedback (ConfirmDialog, ProgressOverlay, Toast, AlertBanner).

O design segue o principio "Honest UI" (NFR-012): sem alarmismo, sem animacoes excessivas, dados neutros e factuais. Estetica gamer moderna com acentos neon discretos.

## Acceptance Criteria
- [x] AC-1: App renderiza em dark theme com fundo `#0D1117` (ou equivalente dark) como default e unico tema
- [x] AC-2: Sidebar fixa exibe links para todos os modulos MVP: Dashboard, Cache Cleaner, GPU Cache, Developer Mode, Gaming Mode, Monitor, Settings
- [x] AC-3: Navegacao entre paginas funciona via TanStack Router sem reload
- [x] AC-4: Componente `AppShell` (`components/layout/AppShell.tsx`) envolve todas as paginas com sidebar + area de conteudo
- [x] AC-5: Componente `ConfirmDialog` exibe modal com descricao da acao, espaco/itens afetados, botoes "Confirmar" e "Cancelar"
- [x] AC-6: Componente `ProgressOverlay` exibe progresso por etapas com barra de progresso e texto de status atual
- [x] AC-7: Componente `Toast` exibe notificacoes nao-bloqueantes (success, error, info) com auto-dismiss em 4s
- [x] AC-8: Componente `AlertBanner` exibe alertas de temperatura/aviso com nivel (warning, critical)
- [x] AC-9: Todas as paginas stub renderizam seu titulo dentro do `AppShell` sem erros
- [x] AC-10: Componentes `Button`, `Toggle`, `Badge`, `Tooltip` implementados com variantes documentadas em comentarios

## Technical Notes

### Cores do dark theme (NFR-003)
```css
/* styles/globals.css */
:root {
  --bg-primary: #0D1117;
  --bg-secondary: #161B22;
  --bg-tertiary: #21262D;
  --border: #30363D;
  --text-primary: #E6EDF3;
  --text-secondary: #8B949E;
  --accent-blue: #58A6FF;
  --accent-cyan: #39D353;
  --accent-green: #3FB950;
  --accent-red: #F85149;
  --accent-yellow: #D29922;
}
```

### Fontes (NFR-003)
- UI geral: Inter (via Google Fonts ou bundlada)
- Dados numericos: JetBrains Mono

### TanStack Router
Usar file-based routing ou configuracao manual. Rotas do MVP:
- `/` → Dashboard
- `/cache` → CacheCleaner
- `/gpu` → GpuCacheCleaner
- `/developer` → DeveloperMode
- `/gaming` → GamingMode
- `/monitor` → Monitor (placeholder para Epic 4)
- `/settings` → Settings

### Sidebar
- Largura fixa: 240px
- Icone SVG simples por modulo (nao usar emoji)
- Item ativo com destaque em `--accent-blue`
- Logo/nome do app no topo

### ConfirmDialog (NFR-004)
```typescript
interface ConfirmDialogProps {
  title: string;
  description: string;
  details?: string; // espaco/itens afetados
  onConfirm: () => void;
  onCancel: () => void;
  dangerous?: boolean; // double confirm para ops DANGEROUS
}
```

### Honest UI (NFR-012)
- Nenhum icone triste ou vermelho para estado neutro/saudavel
- Nenhuma linguagem alarmista nos placeholders de paginas
- Status "Sistema limpo" usa indicador verde neutro, sem animacoes de celebracao
- Nunca usar "RISCO!", "PERIGO!", "SEU PC ESTA LENTO!" como copy
