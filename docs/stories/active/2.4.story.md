## User Story
Como usuario do Windows Optimizer, eu quero ver a lista de categorias de cache com o espaco ocupado por cada uma, selecionar o que quero limpar, e confirmar antes de executar, para que eu tenha controle total sobre o que e deletado.

## Description
Implementar a pagina `CacheCleaner.tsx` completa com: botao de scan, exibicao de resultados por categoria com checkboxes, "Select All" / "Deselect All", estimativa de espaco total a liberar, e botao de limpeza que abre `ConfirmDialog` antes de executar. Seguir o principio Honest UI (NFR-012).

Esta story e puramente frontend — consome os Tauri commands implementados nas Stories 2.1-2.3.

## Acceptance Criteria
- [ ] AC-1: Pagina exibe botao "Scan System Cache" — ao clicar, invoca `scan_system_cache` e exibe spinner durante execucao
- [ ] AC-2: Apos scan, exibe tabela de categorias: [Categoria | Path | Tamanho | Status | Checkbox]
- [ ] AC-3: Cada categoria tem checkbox para inclusao/exclusao da limpeza
- [ ] AC-4: Botoes "Select All" e "Deselect All" funcionam corretamente
- [ ] AC-5: Footer da tabela exibe "Total selecionado: X.X GB" somando apenas categorias marcadas
- [ ] AC-6: Categorias com `requires_elevation: true` exibem badge "Requer Admin" — ao tentar selecionar sem elevacao, exibe explicacao
- [ ] AC-7: Categorias com `browser_running: true` exibem aviso amarelo "Browser aberto — arquivos em uso serao ignorados"
- [ ] AC-8: Botao "Limpar Selecionados" abre `ConfirmDialog` com: lista de categorias, espaco total, botoes "Confirmar" e "Cancelar"
- [ ] AC-9: Apos confirmar, exibe `ProgressOverlay` com progresso de limpeza por categoria
- [ ] AC-10: Apos limpeza, exibe resumo: espaco liberado, arquivos deletados, arquivos ignorados (locked), warnings (ex: "Boot pode ser mais lento")
- [ ] AC-11: Se sistema esta limpo (scan retorna 0 bytes), exibe "Cache limpo" com indicador verde — sem dramatismo (NFR-012)

### Formatacao de bytes (lib/format.ts)
```typescript
export function formatBytes(bytes: number): string {
  if (bytes === 0) return '0 B';
  const k = 1024;
  const sizes = ['B', 'KB', 'MB', 'GB', 'TB'];
  const i = Math.floor(Math.log(bytes) / Math.log(k));
  return `${parseFloat((bytes / Math.pow(k, i)).toFixed(2))} ${sizes[i]}`;
}
```

### Honest UI (NFR-012)
- Estado "sem cache": "Tudo limpo! Nenhum cache encontrado." — texto verde neutro, sem animacao
- Nunca: "Seu sistema esta LENTO!" ou "X mil problemas encontrados!"
- Warnings sobre primeiro boot mais lento: linguagem informativa, nao alarmista

### invoke com hook useInvoke
```typescript
// hooks/useInvoke.ts
export function useInvoke<T>(command: string) {
  return async (args?: Record<string, unknown>): Promise<T> => {
    return invoke<T>(command, args);
  };
}
```
