# Story 3.4: UI do Modulo GPU Cache

## Metadata
- **Epic:** 3 - GPU Cache Cleaner
- **Status:** Done
- **Priority:** P0
- **Story Points:** 3
- **Depends On:** 1.2, 3.1, 3.2, 3.3
- **FR References:** FR-012, FR-016, FR-017, NFR-004, NFR-012

## User Story
Como usuario do Windows Optimizer, eu quero ver as informacoes da minha GPU, os caches de shader disponiveis com estimativa de espaco, e limpa-los com um clique e confirmacao, para que eu gerencie o cache GPU de forma simples e informada.

## Description
Implementar a pagina `GpuCacheCleaner.tsx` com: card de informacoes da GPU detectada, tabela de categorias de cache GPU com tamanhos, aviso nao-bloqueante sobre loading times, selecao de categorias, e fluxo de confirmacao + limpeza com progresso. Pagina puramente frontend consumindo commands das Stories 3.1-3.3.

## Acceptance Criteria
- [x] AC-1: Topo da pagina exibe card com informacoes da(s) GPU(s) detectada(s): nome, vendor, VRAM, driver version
- [x] AC-2: Tabela de cache GPU exibe: [Categoria | Vendor | Paths | Tamanho Total | Disponivel | Checkbox]
- [x] AC-3: Categorias indisponiveis (GPU nao detectada) aparecem com estado disabled e explicacao
- [x] AC-4: Aviso amarelo nao-bloqueante exibido antes da limpeza: "Limpeza de shader cache pode aumentar loading times de jogos temporariamente" (FR-017)
- [x] AC-5: Total selecionado exibido no footer
- [x] AC-6: Botao "Limpar GPU Cache" abre ConfirmDialog com lista de categorias e espaco total
- [x] AC-7: Durante limpeza, ProgressOverlay exibe progresso por categoria
- [x] AC-8: Apos limpeza, resumo exibe espaco liberado e aviso de primeiro boot mais lento
- [x] AC-9: Se NVML nao disponivel (GPU NVIDIA sem driver), exibe informacao em vez de erro
- [x] AC-10: DirectX shader cache sempre aparece disponivel (independente de GPU)

## Technical Notes

### GpuInfo display no card
```typescript
// types/ipc.ts
export interface GpuInfo {
  index: number;
  name: string;
  vendor: string;
  driverVersion: string;
  vramBytes: number;
  nvmlAvailable: boolean;
}
```

### Layout da pagina GpuCacheCleaner.tsx
```
[GPU Info Card(s)]
  GPU: NVIDIA GeForce RTX 3080 | VRAM: 10 GB | Driver: 535.x.x | NVML: Ativo

[Aviso nao-bloqueante - AlertBanner warning]
  "Shader cache sera recompilado ao iniciar jogos apos limpeza. Primeiros minutos podem ter maior loading time."

[Tabela de categorias GPU]
  [checkbox] NVIDIA Shader Cache   | DXCache + GLCache + NV_Cache | 2.3 GB  | Disponivel
  [checkbox] DirectX Shader Cache  | D3DSCache                    | 450 MB  | Disponivel
  [ grayed ] AMD Shader Cache      | DxCache + GLCache            | —       | GPU AMD nao detectada

[Footer]
  Selecionado: 2.75 GB  |  [Limpar GPU Cache Selecionado]
```
