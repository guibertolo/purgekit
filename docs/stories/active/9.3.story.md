# Story 9.3: Limpeza de Caches de Package Managers JS e Docker

## Metadata
- **Epic:** 9 - Developer Mode
- **Status:** Draft
- **Priority:** P0
- **Story Points:** 5
- **Depends On:** 9.1
- **FR References:** FR-055, FR-056

## User Story
Como desenvolvedor, eu quero limpar os caches globais de npm/yarn/pnpm e fazer prune de Docker, para que eu recupere espaco de caches de dependencias que ja foram baixadas e nao sao mais necessarias.

## Description
Implementar scan e limpeza de caches globais de package managers JavaScript (npm, yarn, pnpm) e Docker (images dangling, volumes nao referenciados, build cache). Apenas limpa ferramentas detectadas pela Story 9.1. Docker usa CLI (`docker system prune`) — nao requer elevacao.

## Acceptance Criteria
- [ ] AC-1: Scan de npm cache retorna tamanho de `%APPDATA%\npm-cache\` — apenas se npm detectado (FR-055)
- [ ] AC-2: Scan de yarn cache retorna tamanho de `%LOCALAPPDATA%\Yarn\Cache\` — apenas se yarn detectado
- [ ] AC-3: Scan de pnpm store retorna tamanho de `%LOCALAPPDATA%\pnpm-store\` e `%LOCALAPPDATA%\pnpm\store\` — apenas se pnpm detectado
- [ ] AC-4: Scan de Docker retorna estimativa de espaco via `docker system df` — apenas se Docker detectado e rodando (FR-056)

- [ ] AC-6: Limpeza de npm/yarn/pnpm e delecao direta dos diretorios de cache (mesma logica da Story 2.1)

- [ ] AC-9: Nenhuma das limpezas requer elevacao admin

### Scan de Docker via `docker system df`
```rust
pub fn get_docker_disk_usage() -> Result<DockerDiskUsage, AppError> {
    let output = Command::new("docker")
        .args(["system", "df", "--format", "json"])
        .output()?;

    if !output.status.success() {
        return Err(AppError::Io("Docker nao esta rodando ou nao esta instalado".into()));
    }

    // Parsear output JSON para extrair tamanho recuperavel
    let usage: DockerDiskUsage = serde_json::from_slice(&output.stdout)?;
    Ok(usage)
}
```

Alternativa se `--format json` nao funcionar: usar `docker system df` sem flag e parsear texto.

### Limpeza Docker
```rust
pub fn prune_docker(include_volumes: bool) -> Result<DockerPruneResult, AppError> {
    let mut args = vec!["system", "prune", "-f"];
    if include_volumes {
        args.push("--volumes");
    }

    let output = Command::new("docker").args(&args).output()?;

    Ok(DockerPruneResult {
        success: output.status.success(),
        output: String::from_utf8_lossy(&output.stdout).to_string(),
        freed_bytes: parse_freed_from_output(&output.stdout),
    })
}
```
