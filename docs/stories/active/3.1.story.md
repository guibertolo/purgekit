# Story 3.1: Deteccao de GPU via NVML e WMI

## Metadata
- **Epic:** 3 - GPU Cache Cleaner
- **Status:** Draft
- **Priority:** P0
- **Story Points:** 5
- **Depends On:** 1.3
- **FR References:** FR-012, CON-008

## User Story
Como usuario do Windows Optimizer, eu quero que o app detecte automaticamente minha GPU (NVIDIA, AMD, Intel), para que as funcionalidades de limpeza e monitoramento de GPU sejam ativadas automaticamente para minha placa.

## Description
Implementar `platform/nvml.rs` para deteccao e informacoes de GPU NVIDIA via NVML, e `platform/wmi_client.rs` para deteccao de GPU AMD/Intel via WMI. Expor via `modules/gpu_cleaner.rs` um metodo `detect_gpus()` que retorna informacoes de todas as GPUs instaladas.

Se NVML nao estiver disponivel (drivers NVIDIA nao instalados), fazer graceful degradation: logar aviso e retornar apenas informacoes via WMI. CON-008: AMD monitoring detalhado e para v1.0 — no MVP, apenas deteccao basica via WMI.

## Acceptance Criteria
- [ ] AC-1: Tauri command `detect_gpus` retorna `Vec<GpuInfo>` com informacoes de cada GPU detectada
- [ ] AC-2: Para GPU NVIDIA: `GpuInfo` inclui `name`, `driver_version`, `vram_bytes`, `vendor: "NVIDIA"`, `nvml_available: true`
- [ ] AC-3: Para GPU AMD/Intel: `GpuInfo` inclui `name`, `vendor`, `vram_bytes` (via WMI), `nvml_available: false`
- [ ] AC-4: Se NVML nao estiver disponivel (drivers NVIDIA ausentes), app nao crasha — retorna GPU info via WMI com `nvml_available: false`
- [ ] AC-5: NVML e inicializado uma vez no startup do app e disponibilizado via `AppState` (nao reinicializado a cada query)
- [ ] AC-6: WMI query `Win32_VideoController` retorna nome, driver version e VRAM para todas as GPUs
- [ ] AC-7: `platform/wmi_client.rs` tem funcao `create_connection()` que retorna conexao WMI reutilizavel

## Technical Notes

### platform/nvml.rs
```rust
use nvml_wrapper::Nvml;

pub struct NvmlHandle(Option<Nvml>);

impl NvmlHandle {
    pub fn init() -> Self {
        match Nvml::init() {
            Ok(nvml) => {
                tracing::info!("NVML initialized successfully");
                NvmlHandle(Some(nvml))
            }
            Err(e) => {
                tracing::warn!("NVML not available: {}. NVIDIA features disabled.", e);
                NvmlHandle(None)
            }
        }
    }

    pub fn is_available(&self) -> bool {
        self.0.is_some()
    }

    pub fn get_device_count(&self) -> u32 {
        self.0.as_ref()
            .and_then(|n| n.device_count().ok())
            .unwrap_or(0)
    }
}
```

### Deteccao de vendor
```rust
pub fn detect_vendor(name: &str) -> &'static str {
    let name_lower = name.to_lowercase();
    if name_lower.contains("nvidia") || name_lower.contains("geforce") || name_lower.contains("quadro") {
        "NVIDIA"
    } else if name_lower.contains("amd") || name_lower.contains("radeon") {
        "AMD"
    } else if name_lower.contains("intel") {
        "Intel"
    } else {
        "Unknown"
    }
}
```
