# Story 10.4: Gaming Mode — Ativacao Automatica por Processo e UI

## Metadata
- **Epic:** 10 - Gaming Mode
- **Status:** Draft

- **Depends On:** 10.1, 10.2, 10.3

## User Story
Como usuario gamer, eu quero configurar o executavel do meu jogo para que o Gaming Mode ative e desative automaticamente quando o jogo abrir e fechar, e ter uma interface clara com botao toggle e status visual, para que eu nao precise lembrar de ativar manualmente antes de cada sessao.

## Description
Implementar deteccao automatica de processo do jogo (FR-069): monitor que verifica se o executavel configurado esta rodando e ativa/desativa o Gaming Mode automaticamente. Implementar a pagina `GamingMode.tsx` com: botao toggle proeminente, status visual, lista de jogos configurados, e sumario do que esta ativo.

## Acceptance Criteria

### FR-069 — Ativacao automatica
- [ ] AC-1: Tauri command `add_game_config` aceita `{ name: String, exe_path: String }` e salva na lista persistida em `AppConfig`

- [ ] AC-5: Se Gaming Mode ja ativo manualmente, ativacao automatica nao faz nada (nao re-ativa)

## Technical Notes

### Background task: game_process_watcher
```rust
pub async fn start_game_process_watcher(app: AppHandle) {
    loop {
        tokio::time::sleep(Duration::from_secs(5)).await;

        let state = app.state::<AppState>();
        let config = state.config.lock().unwrap();
        let game_configs = config.game_configs.clone();
        drop(config);

        if state.gaming_mode_active.load(Ordering::Relaxed) {
            // Verificar se o processo do jogo ainda esta rodando
            // Se nao estiver mais, desativar automaticamente
            continue;
        }

        // Verificar se algum jogo configurado esta rodando
        let sys = sysinfo::System::new_all();
        for game in &game_configs {
            let exe_name = Path::new(&game.exe_path)
                .file_name()
                .unwrap_or_default()
                .to_string_lossy();

            if sys.processes_by_name(&exe_name).next().is_some() {
                app.emit("game-detected", &game.name).ok();
                // Ativar Gaming Mode automaticamente
                // (em nova task para nao bloquear o watcher)
                break;
            }
        }
    }
}
```

### Status visual quando ativo (NFR-003 + NFR-012)
Indicador verde pulsante simples (CSS `pulse` animation). Nao usar animacao dramatica. Mostrar lista factual: "Servicos pausados: 5 | RAM liberada: ~420 MB | Game DVR: OFF".

- Flush RAM: `true` (default)
- Desativar Game DVR: `true` (default)
