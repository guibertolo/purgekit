# Story 1.5: Installer NSIS com Deteccao de WebView2

## Metadata
- **Epic:** 1 - Foundation & Tauri Project Setup
- **Status:** Ready for Review
- **Priority:** P1
- **Story Points:** 3
- **Depends On:** 1.1
- **FR References:** NFR-002, NFR-007, CON-004, CON-011

## User Story
Como usuario do Windows Optimizer, eu quero um installer simples que instale o app e verifique se WebView2 esta disponivel, para que eu possa instalar o app facilmente no Windows 10 e Windows 11 sem dependencias manuais.

## Description
Configurar o processo de build do Tauri para gerar um installer NSIS (`.exe`) com menos de 10 MB. O installer deve detectar se WebView2 Runtime esta instalado (Windows 10 pode nao ter) e instalar automaticamente se necessario. Sem bundleware, sem checkboxes extras, sem ofertas — apenas o app (CON-011).

Esta story pode ser parcialmente adiada ao final do epic se bloquear outras stories. O build release do Tauri ja gera NSIS por default — o trabalho principal e configurar `tauri.conf.json` corretamente e testar o tamanho do bundle.

## Technical Notes

### tauri.conf.json — secao bundle
```json
{
  "bundle": {
    "active": true,
    "targets": ["nsis"],
    "identifier": "com.italo.windows-optimizer",
    "publisher": "Italo Gustavo",
    "icon": ["icons/icon.ico"],
    "windows": {
      "webviewInstallMode": {
        "type": "downloadBootstrapper"
      }
    }
  }
}
```

`webviewInstallMode: "downloadBootstrapper"` faz o installer baixar e instalar WebView2 se necessario. Alternativa: `"type": "embedBootstrapper"` embute o bootstrapper no installer (aumenta ~1.5 MB mas funciona offline).

### Tamanho do bundle
O Tauri 2.x gera installers de ~3-5 MB base. Para manter abaixo de 10 MB:
- Nao incluir assets desnecessarios no `tauri.conf.json` (`bundle.resources`)
- Otimizar build Rust com `[profile.release]` no `Cargo.toml`:
  ```toml
  [profile.release]
  opt-level = 3
  lto = true
  codegen-units = 1
  strip = true
  ```
- Otimizar build frontend: `vite build` com minificacao padrao

### CON-004 — sem code signing
SmartScreen pode exibir aviso "Windows protegeu seu PC" ao executar o installer. Para uso pessoal, isso e aceitavel. Nao e necessario implementar code signing.

### build.rs — UAC manifest
Para que o installer possa pedir elevacao, o `build.rs` deve embedar o manifest UAC:
```rust
fn main() {
    tauri_build::build();
    // Windows: embed manifest para UAC
    #[cfg(target_os = "windows")]
    {
        println!("cargo:rerun-if-changed=manifest.xml");
    }
}
```

O Tauri 2 ja lida com o manifest UAC via `tauri.conf.json` → `windows.requestedExecutionLevel`.
